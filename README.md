# find-duplicate
Script ruby for find the duplicates file 

## Usage
```bash
ruby find-duplicate.rb [option] [path]
```

## Options
* -h, --help: Show help
* -l, --low: Low comparison level (faster)
* -d, --deep: Deep comparison level (slower)

## Examples
```bash
ruby find-duplicate.rb -l /home/user
```
